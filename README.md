# 8-Bit-Mario
## Usage
### Install

    npm install 8-bit-mario
    
### Implementation

    ...
    
    import Mario from '8-bit-mario';
    
    ...
    
    class DisplaysMario() {
        constructor() {
            super()
        }
        
        render (
            <Mario></Mario>
        );
    }
